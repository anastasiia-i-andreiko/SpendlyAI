# Spendly AI 💰

Персональний Telegram-бот для автоматичного обліку витрат та управління скарбничкою за допомогою ШІ.
Бот - @SpendAIlyBot.

## Технології
- **Python 3.11**
- **Aiogram 3.x** — для інтерфейсу бота.
- **SQLAlchemy + SQLite** — для зберігання даних.
- **Groq API (Llama 3.3)** — для обробки природної мови (NLP).

## Структура проекту
- `main.py` — запуск бота та обробка команд.
- `database/` — налаштування бази даних та моделі.
- `services/` — бізнес-логіка (ШІ, фінанси, промпти).

## Як запустити
1. Встанови залежності: `pip install aiogram sqlalchemy groq`
2. Додай токени у `config.py`.
3. Запусти: `python main.py`
